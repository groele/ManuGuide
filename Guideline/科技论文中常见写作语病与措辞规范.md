# 科技论文中常见写作语病与措辞规范

> 适用场景：英文科技论文写作、中文科技论文写作、中英混排手稿、摘要与引言、结果讨论、图注与补充信息、投稿前语言润色。  
> 核心原则：**科技论文的写作目标是清晰、准确、简洁。语法错误、冗余表达、逻辑跳跃、中式英语是投稿中常见的语言退回原因。**

---

## 1. 总览：科技论文写作的常见问题

科技论文写作中的常见问题可以归为以下几类：

1. **语法与句法错误**
2. **冗余与啰嗦表达**
3. **逻辑连接与并列结构**
4. **语气与修辞（被动 vs 主动、强调）**
5. **中式英语/Chinglish 典型问题**
6. **图表与正文的引用衔接**
7. **英文摘要与标题的常见问题**
8. **中文科技论文写作规范**

## 2. 语法与句法错误

### 2.1 data 是复数

这是科技论文中最常见也最容易犯的语法错误之一。`data` 是 `datum` 的复数形式，应搭配复数动词。

**正确：**

```text
The data are consistent with the model.
These data clearly show the gate-dependent behavior.
The experimental data were collected at 10 K.
```

**错误：**

```text
The data is consistent with the model.
This data shows the gate-dependent behavior.
```

**注意：** 部分期刊（尤其在计算机科学领域）已开始接受 `data` 作为不可数名词，但在物理论文中建议坚持复数用法。

### 2.2 phenomenon 与 phenomena 的单复数

```text
singular: phenomenon（现象）
plural:   phenomena（多个现象）
```

**正确：**

```text
This phenomenon is attributed to the strain-induced bandgap modulation.
Several interesting phenomena were observed.
```

**错误：**

```text
Several interesting phenomenon were observed.
This phenomena is attributed to...
```

### 2.3 主语与谓语的一致性

**正确：**

```text
The effect of strain and gate voltage on the PL spectrum is significant.
		← 主语是 effect（单数），所以用 is
```

**错误：**

```text
The effect of strain and gate voltage on the PL spectrum are significant.
		← 这里 are 的主语是 effect（单数），不是 strain and gate voltage
```

**正确：**

```text
The PL intensity and the peak position are both modulated by gate voltage.
		← 主语是 "The PL intensity and the peak position"（并列主语，复数）
```

### 2.4 which 从句的逗号

限定性定语从句（restrictive clause）不加逗号，非限定性定语从句（non-restrictive clause）加逗号。

**正确（限定性——不加逗号）：**

```text
The spectra that were measured at low temperature show a narrow linewidth.
		← 强调"那些在低温下测量的"光谱
```

**正确（非限定性——加逗号）：**

```text
The low-temperature spectra, which were measured at 10 K, show a narrow linewidth.
		← 补充说明，即使去掉"which were measured at 10 K"句子仍然成立
```

**常见错误——缺少逗号导致歧义：**

```text
The samples that were annealed showed a stronger PL response.
		← 暗示有些样品没退火，强化效果不同。
		← 如果所有样品都退火了，应去掉 that 从句或加逗号改为非限定性。
```

### 2.5 悬垂修饰语（Dangling modifier）

悬垂修饰语是分词的逻辑主语与主句主语不一致的错误。

**错误：**

```text
Using the dry-transfer method, the heterostructure showed excellent interface quality.
		← "Using" 的逻辑主语应该是"人"，但句子的主语是"heterostructure"
```

**正确：**

```text
Using the dry-transfer method, we fabricated a heterostructure with excellent interface quality.
```

或：

```text
The heterostructure fabricated by the dry-transfer method shows excellent interface quality.
```

**更多示例：**

**错误：**

```text
To measure the PL spectra, a 532 nm laser was used.
		← 谁去测量？"To measure"的逻辑主语不明确
```

**正确：**

```text
To measure the PL spectra, we used a 532 nm laser.
```

## 3. 冗余与啰嗦表达

### 3.1 常见冗余词组

| 冗余表达 | 推荐替换 |
|---------|---------|
| `As is shown in Fig. 1` | `As shown in Fig. 1` |
| `It can be seen that...` | 直接陈述 |
| `It is worth noting that...` | 直接说事实，或用 `Notably, ...` |
| `In order to...` | `To...` |
| `Due to the fact that...` | `Because...` |
| `On the basis of...` | `Based on...` 或 `From...` |
| `A number of...` | `Several...` 或 `Many...` |
| `The majority of...` | `Most...` |
| `In the case of...` | 通常可以去掉 |
| `With respect to...` | `For...` 或 `Regarding...` |
| `In the range of 10–300 K` | `Over 10–300 K` 或 `From 10 to 300 K` |

**推荐：**

```text
As shown in Fig. 1, the PL spectrum shows three peaks.
The PL intensity decreases as the temperature increases.
Notably, the trion peak disappears above 150 K.
```

**不推荐：**

```text
As is shown in Fig. 1, it can be seen that the PL spectrum shows three peaks.
In the case of the temperature-dependent measurement, it was found that the PL intensity decreases.
Due to the fact that the temperature increases, the trion peak disappears.
```

### 3.2 不必要的自我引用

**冗余：**

```text
In this paper, we study the gate-dependent exciton–trion redistribution.
This work investigates the effect of strain on...
```

**更简洁：**

```text
We study the gate-dependent exciton–trion redistribution.
The effect of strain on exciton funneling is investigated...
```

在大多数情况下，读者已经知道他们在读你的论文，不需要反复声明。

### 3.3 重复的动词修饰

**冗余：**

```text
The PL intensity gradually decreased slowly.
The peak position shifted significantly by a large amount.
```

**正确：**

```text
The PL intensity gradually decreased.
The peak position shifted significantly.
```

选择一个修饰词即可，不要叠加。

### 3.4 there is / there are 的滥用

**不推荐：**

```text
There is a clear gate-dependent modulation of the exciton–trion balance.
```

**更简洁：**

```text
The gate voltage clearly modulates the exciton–trion balance.
```

## 4. 逻辑连接与并列结构

### 4.1 while vs whereas

`while` 可表示"同时发生"或"对比"，`whereas` 只表示对比。

**推荐（对比用 whereas 更精确）：**

```text
The XA peak dominates at low temperature, whereas the XT* peak emerges at higher temperatures.
```

**可以（while 表示同时+对比）：**

```text
The PL intensity decreases while the temperature increases.
		← while 同时表达了"随着"和"对比"
```

**注意：** `while` 在科技论文中常用于表示时间关系。如果只表示对比，推荐用 `whereas` 更清晰。

### 4.2 due to vs because of

`due to` 修饰名词，`because of` 修饰动词或句子。

**正确：**

```text
The linewidth broadening is due to the enhanced electron–phonon interaction.
		← due to 修饰名词 "broadening"
The linewidth broadens because of the enhanced electron–phonon interaction.
		← because of 修饰动词 "broadens"
```

### 4.3 并列结构的一致性

并列的若干个成分应在语法形式上保持一致。

**正确：**

```text
The gate voltage modulates the PL intensity, shifts the peak position, and broadens the linewidth.
		← 三个并列动词：modulates, shifts, broadens
```

**错误：**

```text
The gate voltage modulates the PL intensity, shifts the peak position, and the broadening of the linewidth.
		← 前两个是动词，第三个是名词短语，不一致
```

### 4.4 连接词与句子长度

科技论文中应避免过长的复合句。如果一个句子超过 40 词，考虑拆分为两句。

**不推荐：**

```text
The gate-dependent PL spectra of monolayer MoSe2 were measured at 10 K, and the results show that the exciton–trion spectral weight can be continuously tuned from neutral-exciton-dominated to trion-dominated regimes as the gate voltage is swept from −4 to 2.8 V, which is consistent with the electrostatic doping model.
```

**推荐：**

```text
The gate-dependent PL spectra of monolayer MoSe2 were measured at 10 K. The results show that the exciton–trion spectral weight can be continuously tuned from neutral-exciton-dominated to trion-dominated regimes as the gate voltage is swept from −4 to 2.8 V. This behavior is consistent with the electrostatic doping model.
```

## 5. 语气与修辞

### 5.1 被动语态 vs 主动语态

科技论文传统上偏爱被动语态，但现代科技论文越来越多使用主动语态。

**传统被动（仍然广泛使用）：**

```text
The samples were prepared by mechanical exfoliation.
The PL spectra were measured using a 532 nm laser.
```

**现代主动（被越来越多期刊接受，尤其是 Nature 系列）：**

```text
We prepared the samples by mechanical exfoliation.
We measured the PL spectra using a 532 nm laser.
```

**建议：**
- 引言和讨论可以混合使用；
- 方法部分用被动语态更普遍；
- 全文统一风格比"全被动"或"全主动"更重要。

### 5.2 避免过度强调

科技论文中，应通过**逻辑和证据**强调，而不是通过修饰词。

**不推荐：**

```text
This is a very important result.
This highly interesting behavior was observed.
```

**推荐：**

```text
This result demonstrates a direct relationship between local geometry and excitonic spectral weight.
		← 用事实本身体现重要性
```

### 5.3 慎用 always, never, completely 等绝对词

不推荐在科技论文中使用绝对限定词，除非有充分的证据支持。

**不推荐：**

```text
The trion peak always shifts to lower energy.
Strain never affects the exciton binding energy.
```

**更准确：**

```text
Over the measured gate-voltage range, the trion peak shifts to lower energy.
Within our experimental uncertainty, strain does not significantly affect the exciton binding energy.
```

### 5.4 不要在句首用缩写

**错误：**

```text
PL intensity decreases with temperature.
FIG. 1 shows the gate-dependent spectra.
```

**正确：**

```text
The PL intensity decreases with temperature.
Figure 1 shows the gate-dependent spectra.
```

## 6. 中式英语/Chinglish 典型问题

### 6.1 定冠词 the 的缺失或滥用

中文没有冠词系统，所以中国作者经常漏掉或误用 `the`。

**常见缺失：**

```text
Gate voltage was swept from −4 to 2.8 V.
		→ The gate voltage was swept from −4 to 2.8 V.

PL spectrum shows three peaks.
		→ The PL spectrum shows three peaks.
```

**常见滥用：**

```text
The graphene has a high mobility.
		→ Graphene has a high mobility.
	   （物质名词、广义概念不加 the）
```

### 6.2 可数名词与不可数名词

**常见错误：**

```text
Many researches have shown...
		→ Many studies have shown...
	   （research 是不可数名词，没有 researches）

An important equipment...
		→ An important piece of equipment...
	   （equipment 不可数）

A new phenomenon were... → A new phenomenon was...
Several new phenomenon... → Several new phenomena...
```

### 6.3 make + 形容词

**不推荐：**

```text
The strain effect makes the PL peak shift.
```

**更自然：**

```text
The strain effect shifts the PL peak.
Strain induces the PL peak shift.
```

**更多示例：**

```text
This method makes the measurement more accurate.
		→ This method improves the measurement accuracy.

This makes the linewidth narrower.
		→ This narrows the linewidth.
```

### 6.4 with 结构过度使用

**不推荐：**

```text
With the increase of temperature, the PL intensity decreases.
With the decrease of gate voltage, the trion weight increases.
```

**更自然：**

```text
As the temperature increases, the PL intensity decreases.
The trion weight increases as the gate voltage decreases.
```

### 6.5 过度使用 "can be" / "could be"

**不推荐：**

```text
From Fig. 2, it can be seen that the peak position can be modulated.
The linewidth could be found to increase with temperature.
```

**更直接：**

```text
Figure 2 shows that the gate voltage modulates the peak position.
The linewidth increases with temperature.
```

## 7. 图表与正文的引用衔接

### 7.1 图与正文的衔接

**推荐——积极引用：**

```text
Figure 1a shows the PL spectrum of monolayer MoSe₂.
		← 先说图，再说内容
```

**推荐——被动引用：**

```text
The PL spectrum of monolayer MoSe₂ is shown in Fig. 1a.
		← 先说内容，再说出处
```

**不推荐——冗余：**

```text
As shown in Fig. 1a, the PL spectrum shows three peaks.
		← "shows" 重复出现
```

### 7.2 图的编号引用

```text
as shown in Fig. 1
as shown in Figs. 1 and 2
as shown in Fig. 1a–c
as shown in Supplementary Fig. S1
```

**注意：**
- `Fig.` 缩写后加句点；
- `Figs.` 是复数缩写；
- 句首用 `Figure`。

### 7.3 表的引用

```text
Table 1 summarizes the fitting parameters.
The fitting parameters are listed in Table 1.
As listed in Table 1, ...
```

## 8. 英文摘要与标题的常见问题

### 8.1 摘要中的时态

```text
Background:   present tense → Monolayer MoSe₂ is a promising platform for...
This work:    past tense   → We fabricated h-BN-encapsulated devices and measured...
Results:      present tense → The results demonstrate that...
```

### 8.2 摘要不宜出现的表述

**不推荐：**

```text
In this paper, we report...
		← 删掉 "In this paper"，读者知道你在讲你的论文
First-principles calculations were performed...
		← 如果"进行计算"是核心结果可以保留，但通常应强调"结果"而非"计算行为"
The results will be discussed.
		← 摘要中不要提"将要讨论"，直接说结果
```

### 8.3 标题不宜过长

**推荐：**

```text
Gate-tunable exciton–trion redistribution in monolayer MoSe₂
		← 13 词，信息密度高
```

**不推荐：**

```text
A study on the gate-tunable exciton–trion redistribution phenomenon observed in monolayer MoSe₂
		← 冗余："A study on"、"phenomenon observed" 都可删
```

## 9. 中文科技论文写作规范

### 9.1 中文术语的一致性

同一概念在整篇论文中应使用同一译名。

```text
激子（不混用"激子"和"受激子"）
带电激子（不混用"带电激子"和"三子"或"trion"）
```

### 9.2 中文中的英文术语

如果英文术语比中文更常用，可以直接使用英文。

```text
PL 光谱（而非"光致发光光谱"）
gate voltage 调控（而非"栅极电压调控"）
stacking 构型（而非"堆垛构型"）
```

### 9.3 中文中的"的"字问题

中文科技论文中"的"字容易过度使用。

**不推荐：**

```text
栅压对激子的光谱特性的调控效果
```

**推荐：**

```text
栅压对激子光谱特性的调控
```

### 9.4 中文科技论文的标点

- 中文中用中文标点（逗号、句号、括号等）；
- 英文术语之间用英文标点；
- 中英文混排的段落中，一般统一使用中文标点。

## 10. 常见错误清单

### 10.1 data is / data are

**错误：** `The data is consistent...`
**正确：** `The data are consistent...`

### 10.2 多余冠词

**错误：** `The graphene has a high mobility.`
**正确：** `Graphene has a high mobility.`

### 10.3 悬垂修饰语

**错误：** `Using the dry-transfer method, the heterostructure showed...`
**正确：** `Using the dry-transfer method, we fabricated...`

### 10.4 冗余开头

**错误：** `In this paper, we study the...`
**推荐：** `We study the...`

### 10.5 重复 "shows"

**错误：** `As shown in Fig. 1, the spectrum shows three peaks.`
**正确：** `Figure 1 shows three peaks.` 或 `As shown in Fig. 1, the spectrum exhibits three peaks.`

### 10.6 句首缩写

**错误：** `PL intensity decreases with temperature.`
**正确：** `The PL intensity decreases with temperature.`

## 11. 投稿前检查清单

投稿前建议逐项检查：

### 11.1 全文搜索常见语法错误

```text
搜索 data is / data are
搜索 phenomenon / phenomena 混用
搜索 researches / evidences（不可数名词是否被错误加 s）
```

### 11.2 检查 there is / there are 开头的句子

```text
搜索 "There is" 和 "There are"
		← 至少一半可以重写为更直接的句子
```

### 11.3 检查冗余表达

```text
搜索 "As is shown" → 改为 "As shown"
搜索 "It can be seen" → 改为直接陈述
搜索 "In order to" → 改为 "To"
搜索 "Due to the fact" → 改为 "Because"
```

### 11.4 检查句首

```text
搜索句首的缩写 → 加冠词或改句子结构
搜索句首的 "In this paper" / "In this work" → 删除
```

### 11.5 检查主语谓语一致性

```text
搜索 "effect of ... are" → 检查是否应为 "is"
搜索 "results of ... is" → 检查是否应为 "are"
```

### 11.6 中文论文检查

```text
全文术语是否统一
"的"字是否过度使用
中英文标点是否混用
```

## 12. 最简速记规则

如果只能记住十条：

1. **data 是复数：** `The data are...`
2. **phenomena 是复数：** `This phenomenon...` / `These phenomena...`
3. **删除冗余：** `As shown` 不要 `As is shown`
4. **主动语态更现代：** `We measured...` 而非 `It was measured...`
5. **不是"in this paper"：** 直接说结果
6. **句首不用缩写：** `Figure` 而非 `Fig.`
7. **因短语更有力：** `as T increases` 而非 `with the increase of T`
8. **避免绝对词：** 慎用 `always`, `never`, `completely`
9. **平行结构：** 并列成分语法一致
10. **中英文术语统一：** 全文使用同一译名

---

## 13. 建议的全文统一方案

| 场景 | 统一方案 |
|------|---------|
| data 动词形式 | 复数：`The data are...` |
| 语态 | 方法用被动，结果用主动，全文一致 |
| 图表引用 | 句首 `Figure`，句中 `Fig.` |
| 句子长度 | 不超过 40 词，拆分为短句 |
| 冗余词组 | 执行搜索替换 |
| 中式英语 | 重点检查冠词、with 结构、make 结构 |
| 中文术语 | 全文统一译名 |
| 英文术语 | 正文中使用标准缩写 |

---

## 14. 最后建议

科技论文的语言不是装饰，而是**科学表达的第一载体**：

- 一个语法错误会降低审稿人对文章整体质量的信任；
- 冗余表达会让审稿人觉得作者写作经验不足；
- 中式英语虽然不影响科学内容，但会增加审稿人的理解负担；
- 图表引用和句子结构的一致性使论文更加专业。

投稿前，建议将语言润色作为一个独立项目处理，尤其重点检查：
1. 语法和数据的一致性；
2. 冗余表达和语气；
3. 图表引用的流畅性；
4. 中英文术语的统一性；
5. 摘要和标题的简洁性。
