# 科技论文中斜体的使用规范

> 适用场景：英文科技论文、中文科技论文、中英混排手稿、图注、补充信息、公式、参考文献、Word/LaTeX 排版校对。  
> 核心原则：**斜体主要用于变量、物理量符号、数学未知量、可变指标、拉丁学名等；正体主要用于单位、化学式、函数名、描述性标签、缩写和普通文本。**

---

## 1. 总览：什么时候使用斜体？

在科技论文中，斜体通常用于以下场景：

1. **数学变量和物理量符号**
2. **坐标变量和函数自变量**
3. **可变的下标或上标**
4. **统计学变量和参数**
5. **基因名称、物种拉丁学名等特定学科命名**
6. **参考文献中的期刊名或书名**
7. **少量需要强调的词，但科技论文中应慎用**

不应使用斜体的常见场景包括：

1. **单位**
2. **化学元素符号和化学式**
3. **数学函数名**
4. **数学常数和运算符**
5. **描述性下标**
6. **仪器、方法、材料缩写**
7. **图、表、方程编号**
8. **普通英文缩写**

---

## 2. 最重要的区分：变量斜体，单位正体

### 2.1 变量和物理量符号使用斜体

物理量、变量、参数通常使用斜体。

**推荐：**

```text
T = 300 K
Vg = −4 V
E = 1.65 eV
I = 0.25 a.u.
B = 9 T
d = 100 nm
h = 50 nm
```

在正式排版中应写为：

```latex
$T = 300~\mathrm{K}$
$V_\mathrm{g} = -4~\mathrm{V}$
$E = 1.65~\mathrm{eV}$
$I = 0.25~\mathrm{a.u.}$
$B = 9~\mathrm{T}$
$d = 100~\mathrm{nm}$
$h = 50~\mathrm{nm}$
```

其中：
- `T, V, E, I, B, d, h` 是变量或物理量，使用斜体；
- `K, V, eV, a.u., T, nm` 是单位，使用正体。

---

### 2.2 单位必须使用正体

单位不是变量，因此不能斜体。

**正确：**

```text
300 K
532 nm
0.1 mW
1.65 eV
9 T
10 μm
```

**不推荐：**

```text
300 K
532 nm
0.1 mW
1.65 eV
```

如果在 Word 中，注意不要把单位跟随变量一起设成斜体。

**错误示例：**

```text
T = 300 K
```

如果整段公式都斜体，`K` 会错误地变成斜体。

**推荐：**

```text
T = 300 K
```

其中 `T` 斜体，`K` 正体。

---

## 3. 数学公式中的斜体规则

---

### 3.1 单字母变量使用斜体

数学变量、未知量和物理量通常使用斜体。

```text
x, y, z
r, t
E, F, P
T, V, I
n, k, q
```

**示例：**

```latex
$F = ma$
$E = h\nu$
$U_X(r) = E_X(r) - E_X^0$
```

---

### 3.2 多字母函数名使用正体

函数名不是多个变量相乘，因此应使用正体。

**正确：**

```latex
\sin x
\cos \theta
\exp(-x)
\ln x
\log x
\max(x)
\min(x)
```

**不推荐：**

```latex
sin x
cos θ
exp(-x)
ln x
```

在 LaTeX 中应使用：

```latex
$\sin x$
$\cos \theta$
$\exp(-x)$
$\ln x$
$\log x$
```

而不是：

```latex
$sin x$
$cos \theta$
$exp(-x)$
```

---

### 3.3 数学常数建议使用正体

严格的科技排版中，数学常数通常使用正体，尤其是：
- 自然常数 `e`
- 虚数单位 `i`
- 圆周率 `π`
- 微分符号 `d`

**推荐：**

```latex
\mathrm{e}^{-x}
\mathrm{i}
\pi
\mathrm{d}x
```

**示例：**

```latex
I = I_0 \mathrm{e}^{-\alpha x}
```

```latex
\int f(x)\,\mathrm{d}x
```

```latex
\psi = A\mathrm{e}^{\mathrm{i}kx}
```

**说明：**
- 很多物理期刊对 `π` 是否正体并不强制；
- `e`、`i`、`\mathrm{d}` 使用正体是更严格、更规范的写法；
- 在普通论文中，LaTeX 默认数学斜体也常见，但精排时建议统一。

---

### 3.4 运算符使用正体

运算符不是变量，应使用正体。

```latex
\lim
\sum
\prod
\int
\nabla
\partial
\operatorname{Tr}
\operatorname{Re}
\operatorname{Im}
```

**示例：**

```latex
\operatorname{Re}[\sigma(\omega)]
```

```latex
\operatorname{Tr}(\rho)
```

---

## 4. 下标和上标：最容易出错的部分

下标是否斜体，取决于它的含义。

> **判断原则：如果下标是一个变量或可变指标，用斜体；如果下标是描述性标签、缩写或固定名称，用正体。**

---

### 4.1 描述性下标用正体

描述性下标表示名称、类型、物理意义或缩写，不是变量。

**推荐：**

```latex
E_\mathrm{g}
E_\mathrm{F}
k_\mathrm{B}
\mu_\mathrm{B}
V_\mathrm{g}
I_\mathrm{PL}
I_\mathrm{dark}
I_\mathrm{on}
I_\mathrm{off}
T_\mathrm{c}
T_\mathrm{m}
R_\mathrm{s}
R_\mathrm{sh}
```

对应含义：

| 符号 | 含义 | 推荐原因 |
|---|---|---|
| `E_\mathrm{g}` | bandgap energy | `g` 表示 gap，是描述性标签 |
| `E_\mathrm{F}` | Fermi energy | `F` 表示 Fermi，是专名/标签 |
| `k_\mathrm{B}` | Boltzmann constant | `B` 表示 Boltzmann |
| `\mu_\mathrm{B}` | Bohr magneton | `B` 表示 Bohr |
| `V_\mathrm{g}` | gate voltage | `g` 表示 gate |
| `I_\mathrm{PL}` | PL intensity | `PL` 是缩写 |
| `T_\mathrm{c}` | critical temperature | `c` 表示 critical |
| `R_\mathrm{s}` | series resistance / sheet resistance | `s` 是描述性标签 |

---

### 4.2 可变指标下标用斜体

如果下标代表一个可变的数学指标、方向、序号或粒子编号，则使用斜体。

**推荐：**

```latex
x_i
E_n
\psi_m
a_j
r_i
k_x
k_y
q_i
```

其中：
- `i, j, m, n` 是指标；
- `x, y` 表示坐标方向或变量分量，通常斜体。

---

### 4.3 描述性标签和变量指标混合出现

**示例：**

```latex
I_{\mathrm{PL},i}
E_{\mathrm{g},n}
V_{\mathrm{g},i}
A_{X_A,i}
```

含义：
- `PL`, `g`, `X_A` 是描述性标签；
- `i`, `n` 是可变指标。

---

### 4.4 物理态标签通常用正体

如果下标表示物理态、峰名称、材料相、模式名称，通常用正体。

**推荐：**

```latex
X_\mathrm{A}
X_\mathrm{T}^{\ast}
X_\mathrm{L}
E_{X_\mathrm{A}}
I_{X_\mathrm{T}^{\ast}}
A_{X_\mathrm{A}}
```

**说明：**
- `X` 如果作为 exciton 符号，可以保留数学斜体，也可以作为峰标签使用正体，关键是全文统一；
- `A`, `T`, `L` 如果分别表示 neutral exciton, trion-related emission, localized emission，则推荐作为标签用正体；
- 星号 `\ast` 不需要斜体。

对于你的激子光谱论文，更推荐统一写法：

```latex
X_\mathrm{A}
X_\mathrm{T}^{\ast}
X_\mathrm{L}
I_{X_\mathrm{A}}/I_{X_\mathrm{T}^{\ast}}
S_\mathrm{T}
```

其中：
- `I`、`S` 是物理量，斜体；
- `X_\mathrm{A}`、`X_\mathrm{T}^{\ast}` 是峰标签；
- `T` 如果表示 trion weight 的标签，建议用正体：`S_\mathrm{T}`。

---

### 4.5 上标的正斜体

上标也遵循相同原则。

**变量指数用斜体：**

```latex
x^n
E^m
```

**描述性上标用正体：**

```latex
X_\mathrm{T}^{-}
X_\mathrm{T}^{+}
E^\mathrm{calc}
E^\mathrm{exp}
I^\mathrm{norm}
```

**注意：**
- 电荷符号 `+`、`−` 本身不是斜体；
- `calc`, `exp`, `norm` 是描述性标签，应使用正体。

---

## 5. 矢量、矩阵和张量的斜体规则

---

### 5.1 矢量通常用粗斜体

物理学和工程论文中，矢量常用粗斜体表示。

**推荐：**

```latex
\boldsymbol{r}
\boldsymbol{k}
\boldsymbol{F}
\boldsymbol{E}
```

**示例：**

```latex
\boldsymbol{F}_X = -\nabla U_X(\boldsymbol{r})
```

---

### 5.2 单位矢量可以用帽符号或粗体

```latex
\hat{\boldsymbol{x}}
\hat{\boldsymbol{y}}
\hat{\boldsymbol{z}}
\hat{\boldsymbol{e}}_r
```

---

### 5.3 矩阵和张量

矩阵和张量的格式因期刊而异，但常见做法包括：

```latex
\mathbf{A}
\mathbf{M}
\boldsymbol{\sigma}
\boldsymbol{\varepsilon}
```

其中：
- 拉丁字母矩阵可用粗正体 `\mathbf{A}`；
- 希腊字母张量可用粗斜体 `\boldsymbol{\sigma}`；
- 关键是全文统一。

---

## 6. 物理量和常数中的斜体规范

---

### 6.1 常见物理量

| 物理量 | 推荐写法 | 说明 |
|---|---|---|
| 温度 | `T` | 斜体 |
| 电压 | `V` | 斜体 |
| 栅压 | `V_\mathrm{g}` | `V` 斜体，`g` 正体 |
| 电流 | `I` | 斜体 |
| 光强 | `I_\mathrm{PL}` | `PL` 正体 |
| 能量 | `E` | 斜体 |
| 带隙 | `E_\mathrm{g}` | `g` 正体 |
| 费米能级 | `E_\mathrm{F}` | `F` 正体 |
| 磁场 | `B` | 斜体 |
| 波矢 | `k` 或 `\boldsymbol{k}` | 标量斜体，矢量粗斜体 |
| 频率 | `f` 或 `\nu` | 斜体 |
| 角频率 | `\omega` | 斜体 |
| 波长 | `\lambda` | 斜体 |
| 电荷量 | `q` | 斜体 |
| 电子电荷 | `e` | 若表示常数可正体，若作为变量可斜体 |
| 介电常数 | `\varepsilon` | 斜体 |
| 极化强度 | `P` | 斜体 |
| 应变 | `\varepsilon` | 斜体 |
| 应力 | `\sigma` | 斜体 |

---

### 6.2 常见物理常数

| 常数 | 推荐写法 | 说明 |
|---|---|---|
| Boltzmann constant | `k_\mathrm{B}` | `k` 斜体，`B` 正体 |
| Planck constant | `h` | 斜体常见 |
| Reduced Planck constant | `\hbar` | 斜体/符号体 |
| Bohr magneton | `\mu_\mathrm{B}` | `B` 正体 |
| Vacuum permittivity | `\varepsilon_0` | `0` 非字母，无斜体问题 |
| Elementary charge | `e` 或 `\mathrm{e}` | 建议与数学常数 `e` 区分 |
| Speed of light | `c` | 斜体常见 |

---

## 7. 化学、材料和晶体学中的斜体

---

### 7.1 化学元素和化学式不用斜体

化学元素符号、化学式、材料名称一般使用正体。

**正确：**

```text
MoSe₂
WSe₂
ReS₂
h-BN
SiO₂/Si
Au
graphene
CuInP₂S₆
```

**不推荐：**

```text
MoSe₂
WSe₂
ReS₂
Au
```

在 Word 中要特别注意：如果整句设置为斜体，化学式也会错误变成斜体。

---

### 7.2 化学式中的下标数字不是斜体

```text
MoSe₂
SiO₂
H₂O
CO₂
```

数字下标应为正体数字，不涉及斜体。

---

### 7.3 晶面、晶向和晶系

晶体学符号通常有自己的格式规范。

| 类型 | 示例 | 常见写法 |
|---|---|---|
| 晶面 | `(001)` | 正体数字 |
| 晶向 | `[100]` | 正体数字 |
| 晶面族 | `{100}` | 正体数字 |
| 晶向族 | `<100>` | 正体数字 |
| Miller 指数变量 | `(hkl)` | `h, k, l` 可作为变量斜体 |

**示例：**

```latex
(001)
[100]
(hkl)
```

其中 `(hkl)` 中的 `h, k, l` 表示变量指标，可以斜体。

---

### 7.4 相、结构和堆垛标签

相名、堆垛构型、样品标签通常不用斜体。

```text
1T phase
2H phase
AA stacking
AB stacking
AB′XY stacking
R phase
H phase
```

如果期刊或领域习惯将某些相符号斜体，应遵循目标期刊格式，但默认不建议随意斜体。

---

## 8. 生物和化学命名中的斜体

---

### 8.1 物种拉丁学名使用斜体

属名和种加词使用斜体。

```text
Escherichia coli
Homo sapiens
Drosophila melanogaster
```

属名首字母大写，种加词小写。

---

### 8.2 首次出现后可缩写

```text
Escherichia coli
E. coli
```

其中 `E.` 和 `coli` 仍使用斜体。

---

### 8.3 更高分类阶元通常不用斜体

```text
Mammalia
Proteobacteria
Asteraceae
```

这些通常不用斜体，具体依领域和期刊要求而定。

---

### 8.4 基因名称常用斜体，蛋白名称通常不用斜体

在生物医学论文中：

```text
BRCA1
TP53
lacZ
```

基因名常使用斜体。

蛋白名通常不用斜体：

```text
BRCA1 protein
p53 protein
```

注意：具体格式因物种和期刊差异很大，应以目标期刊或数据库规范为准。

---

## 9. 拉丁词、外来词和缩写

---

### 9.1 常见拉丁缩写通常不需要斜体

现代科技论文中，常见拉丁缩写通常使用正体。

```text
et al.
i.e.
e.g.
vs.
in situ
in vitro
in vivo
ex situ
a priori
ad hoc
```

**说明：**
- 很多旧式排版会将部分拉丁词斜体；
- 现代期刊通常不要求斜体；
- 最重要的是遵循目标期刊风格并全文统一。

---

### 9.2 未被英语吸收的外来词可斜体

如果某个外来词不常见，且期刊风格允许，可以斜体。

但科技论文中通常不建议为了修辞效果频繁使用斜体外来词。

---

## 10. 正文中的强调：科技论文应慎用斜体

斜体可以用于强调，但科技论文中不推荐频繁使用。

**不推荐：**

```text
This result is very important for excitonic computing.
```

**更推荐：**

```text
This result demonstrates that excitonic spectral weights can serve as programmable physical variables for optoelectronic computing.
```

科学写作中，应通过逻辑和措辞强调，而不是依赖斜体视觉强调。

---

## 11. 图、表、公式编号中的斜体

---

### 11.1 Fig., Table, Eq. 通常不用斜体

**正确：**

```text
Fig. 1a
Fig. 2a–d
Table 1
Eq. (1)
Supplementary Fig. S3
```

**不推荐：**

```text
Fig. 1a
Table 1
Eq. (1)
```

不要把 `Fig.`、`Table`、`Eq.` 设置为斜体，除非期刊模板有特殊要求。

---

### 11.2 图中的坐标轴变量

坐标轴中的物理量符号可斜体，单位正体。

**推荐：**

```text
Energy, E (eV)
Gate voltage, Vg (V)
Intensity, IPL (a.u.)
Magnetic field, B (T)
Temperature, T (K)
```

正式排版建议：

```latex
Energy, $E$ (eV)
Gate voltage, $V_\mathrm{g}$ (V)
PL intensity, $I_\mathrm{PL}$ (a.u.)
Magnetic field, $B$ (T)
Temperature, $T$ (K)
```

---

### 11.3 图例中的变量和标签

**推荐：**

```latex
$X_\mathrm{A}$
$X_\mathrm{T}^{\ast}$
$S_\mathrm{T}$
$V_\mathrm{g}=2.8~\mathrm{V}$
```

**不推荐：**

```text
XA
XT*
ST
Vg=2.8V
```

---

## 12. 参考文献中的斜体

参考文献格式取决于期刊，但常见规则如下。

---

### 12.1 期刊名通常使用斜体

```text
Nature
Nature Communications
Physical Review Letters
Advanced Materials
ACS Nano
```

在参考文献列表中，期刊名通常为斜体，卷号可能加粗，页码或文章号正体。

**示例：**

```text
Smith, J. et al. Title of the article. Nature 600, 123–130 (2021).
```

实际排版中：
- 期刊名 `Nature` 常为斜体；
- 卷号 `600` 常为加粗；
- 文章题名通常不用斜体。

---

### 12.2 书名通常使用斜体

```text
Introduction to Solid State Physics
Principles of Condensed Matter Physics
```

---

### 12.3 文章标题通常不用斜体

大多数科技期刊的参考文献中，文章标题不用斜体。

---

## 13. 中文科技论文中的斜体

---

### 13.1 中文正文一般不使用斜体

中文科技论文中，中文汉字通常不使用斜体。  
如果需要强调，优先通过措辞、句式或加粗，而不是斜体。

**不推荐：**

```text
该结果说明应变调控非常重要。
```

**更推荐：**

```text
该结果表明，应变调控是决定局域激子响应的关键因素。
```

---

### 13.2 中文中夹杂英文变量

中文句子中的变量仍遵循英文科技排版规则。

**推荐：**

```text
随着栅压 V_g 从 −4 V 增加到 2.8 V，S_T 呈现单调增加趋势。
```

正式排版建议：

```latex
随着栅压 $V_\mathrm{g}$ 从 $-4~\mathrm{V}$ 增加到 $2.8~\mathrm{V}$，$S_\mathrm{T}$ 呈现单调增加趋势。
```

---

### 13.3 中文中材料名不用斜体

```text
MoSe₂ 单层
h-BN 封装
Au 微柱
SiO₂/Si 衬底
```

材料名和化学式均应保持正体。

---

## 14. 中英混排中的斜体规范

---

### 14.1 英文术语不因出现在中文中而自动斜体

**正确：**

```text
本文通过 gate-dependent PL spectroscopy 研究激子–带电激子光谱权重重分布。
```

其中 `gate-dependent PL spectroscopy` 是普通英文术语，不需要斜体。

---

### 14.2 英文变量仍应斜体

**推荐：**

```text
激子势可表示为 U_X(r) = E_X(r) − E_X^0。
```

正式排版：

```latex
激子势可表示为 $U_X(r) = E_X(r) - E_X^0$。
```

---

### 14.3 英文缩写不用斜体

```text
PL
SHG
Raman
AFM
SEM
TEM
DFT
MC
CNN
SVM
PCA
```

这些是方法、仪器或模型缩写，通常使用正体。

---

## 15. 科技论文中常见符号的推荐写法

以下是材料物理、二维材料、激子光谱论文中常用符号的推荐格式。

| 含义 | 推荐写法 | 说明 |
|---|---|---|
| 栅压 | `$V_\mathrm{g}$` | `g` 表示 gate，正体 |
| 漏极电压 | `$V_\mathrm{d}$` | `d` 表示 drain，正体 |
| 源漏电压 | `$V_\mathrm{ds}$` | `ds` 表示 drain–source，正体 |
| PL 强度 | `$I_\mathrm{PL}$` | `PL` 是缩写，正体 |
| 中性激子峰 | `$X_\mathrm{A}$` | `A` 是峰标签，正体 |
| 带电激子相关峰 | `$X_\mathrm{T}^{\ast}$` | `T` 是峰标签，正体 |
| 局域发射峰 | `$X_\mathrm{L}$` | `L` 是峰标签，正体 |
| 激子峰积分 | `$A_{X_\mathrm{A}}$` | `A` 为面积变量，峰标签为正体 |
| trion spectral weight | `$S_\mathrm{T}$` | `T` 是 trion 标签 |
| 激子势 | `$U_X(r)$` | `U` 和 `r` 斜体，`X` 可按标签统一 |
| 激子力 | `$\boldsymbol{F}_X$` | 矢量粗斜体 |
| 带隙 | `$E_\mathrm{g}$` | `g` 正体 |
| 费米能级 | `$E_\mathrm{F}$` | `F` 正体 |
| Zeeman splitting | `$\Delta E$` | `E` 斜体 |
| Bohr magneton | `$\mu_\mathrm{B}$` | `B` 正体 |
| g factor | `$g$ factor` 或 `$g$-factor` | `g` 斜体 |
| 磁场 | `$B$` | 斜体 |
| 温度 | `$T$` | 斜体 |
| 柱直径 | `$D$` | 斜体 |
| 抬升高度 | `$h$` | 斜体 |
| 长宽比 | `$h/d$` | 变量斜体 |

---

## 16. 推荐写法示例：二维材料激子论文

---

### 16.1 正文句子示例

**推荐：**

```text
The gate voltage Vg was swept from −4 to 2.8 V to modulate the exciton–trion spectral balance.
```

更正式排版：

```latex
The gate voltage $V_\mathrm{g}$ was swept from $-4$ to $2.8~\mathrm{V}$ to modulate the exciton--trion spectral balance.
```

---

### 16.2 光谱分峰示例

```latex
I_\mathrm{norm}(E)=B(E)+\sum_i A_i
\exp\left[-\frac{(E-E_i)^2}{2\sigma_i^2}\right]
```

说明：
- `I`, `E`, `B`, `A`, `i`, `\sigma` 是变量或指标；
- `norm` 是描述性标签，正体；
- `exp` 是函数名，正体；
- 微分、单位、描述性标签不要斜体。

---

### 16.3 激子–带电激子比值

```latex
R_{A/T}=\frac{A_{X_\mathrm{A}}}{A_{X_\mathrm{T}^{\ast}}}
```

更推荐使用描述清楚的形式：

```latex
R_{X_\mathrm{A}/X_\mathrm{T}^{\ast}}
=
\frac{A_{X_\mathrm{A}}}{A_{X_\mathrm{T}^{\ast}}}
```

---

### 16.4 trion spectral weight

```latex
S_\mathrm{T}
=
\frac{A_{X_\mathrm{T}^{\ast}}}
{A_{X_\mathrm{T}^{\ast}}+A_{X_\mathrm{A}}}
```

说明：
- `S` 是定义的谱权重变量，斜体；
- `T` 表示 trion 标签，正体；
- `A` 是积分面积变量，斜体；
- 峰标签使用正体。

---

### 16.5 Zeeman splitting

```latex
\Delta E = E_{\sigma^+}-E_{\sigma^-}=g\mu_\mathrm{B}B
```

说明：
- `E`, `g`, `B` 是变量或物理量，斜体；
- `\mu_\mathrm{B}` 中的 `B` 表示 Bohr，正体；
- `\sigma^+`, `\sigma^-` 是偏振标签，建议全文统一格式。

---

## 17. Word 中的实操建议

---

### 17.1 不要整段斜体公式说明

很多错误来自于将整段公式说明全部斜体，导致单位、标签、缩写也被错误斜体。

**错误倾向：**

```text
Vg = −4 V, IPL = 0.5 a.u.
```

如果整段都斜体，`V` 单位和 `PL` 标签都会错误。

**正确做法：**
- 只将变量斜体；
- 单位、缩写、描述性下标保持正体；
- 使用 Word 公式编辑器或 MathType 更安全。

---

### 17.2 Word 公式编辑器

在 Word 公式编辑器中：
- 单字母变量通常自动斜体；
- 函数名如 `sin`, `cos`, `exp` 通常自动正体；
- 普通英文标签可能被自动斜体，需要手动设置为正体。

例如输入：

```text
V_g
```

可能显示为 `Vg` 且 `g` 斜体。  
如果 `g` 表示 gate，建议改为：

```text
V_\mathrm{g}
```

在 Word 中可通过公式的“普通文本”或正体设置实现。

---

### 17.3 Word 校对重点

全文搜索并检查：

```text
Vg
Eg
EF
kB
μB
IPL
XA
XT
ST
```

建议统一改为：

```text
V_g
E_g
E_F
k_B
μ_B
I_PL
X_A
X_T*
S_T
```

在最终排版中进一步设置：
- 主变量斜体；
- 描述性下标正体；
- 单位正体；
- 化学式正体。

## 19. 常见错误清单

---

### 19.1 单位被斜体

**错误：**

```text
T = 300 K
λ = 532 nm
P = 0.1 mW
```

如果 `K`、`nm`、`mW` 被设为斜体，就是错误。

**正确：**

```latex
$T=300~\mathrm{K}$
$\lambda=532~\mathrm{nm}$
$P=0.1~\mathrm{mW}$
```

---

### 19.2 描述性下标被斜体

**错误：**

```latex
$V_g$
$E_g$
$E_F$
$I_PL$
```

**正确：**

```latex
$V_\mathrm{g}$
$E_\mathrm{g}$
$E_\mathrm{F}$
$I_\mathrm{PL}$
```

---

### 19.3 化学式被斜体

**错误：**

```text
MoSe₂
h-BN
Au
SiO₂
```

如果这些材料名被斜体，应改为正体。

---

### 19.4 函数名被斜体

**错误：**

```latex
$exp(-x)$
$sin \theta$
$log x$
```

**正确：**

```latex
$\exp(-x)$
$\sin\theta$
$\log x$
```

---

### 19.5 用斜体强调普通词

**不推荐：**

```text
This is very important.
```

**推荐：**

```text
This result is critical because it establishes a direct mapping between local geometry and excitonic spectral weight.
```

---

### 19.6 缩写被斜体

**错误：**

```text
PL
SHG
Raman
DFT
CNN
```

如果这些方法缩写被斜体，通常应改为正体。

## 21. 最简速记规则

如果只能记住十条：

1. **变量斜体：** `E`, `T`, `V`, `I`, `B`, `x`, `t`
2. **单位正体：** `K`, `V`, `eV`, `nm`, `mW`
3. **函数正体：** `sin`, `cos`, `exp`, `log`, `ln`
4. **描述性下标正体：** `V_\mathrm{g}`, `E_\mathrm{g}`, `I_\mathrm{PL}`
5. **可变指标斜体：** `x_i`, `E_n`, `A_i`
6. **化学式正体：** `MoSe₂`, `h-BN`, `Au`
7. **缩写正体：** `PL`, `SHG`, `DFT`, `CNN`
8. **矢量粗斜体：** `\boldsymbol{r}`, `\boldsymbol{k}`, `\boldsymbol{F}`
9. **物种拉丁学名斜体：** `Escherichia coli`
10. **不要用斜体代替科学逻辑强调。**

|---|---|
| 物理量 | 斜体 |
| 单位 | 正体 |
| 描述性下标 | 正体 |
| 可变指标 | 斜体 |
| 化学式 | 正体 |
| 方法缩写 | 正体 |
| 数学函数 | 正体 |

## 23. 最后建议

科技论文中的斜体本质上不是“美观问题”，而是**语义编码问题**：

- 斜体表示变量、可变参数或特定命名；
- 正体表示单位、标签、函数、缩写、材料名和普通文本；
- 下标是否斜体取决于它是“变量”还是“描述性标签”；
- 图注、坐标轴和补充信息中的斜体错误最常见，也最容易影响论文专业感。

投稿前，建议将斜体作为一个独立校对项目处理，尤其重点检查：
1. 变量与单位；
2. 描述性下标；
3. 化学式；
4. 函数名；
5. 图注和坐标轴；
6. 参考文献格式；
7. 中英混排中的变量和英文缩写。
