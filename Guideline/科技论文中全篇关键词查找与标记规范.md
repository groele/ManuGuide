# 科技论文中全篇关键词查找与标记功能需求文档

## 一、功能概述

本功能为 Word VSTO 插件（Manuscript Guide）的扩展模块，旨在帮助用户在科技论文写作过程中，对整篇文档进行关键词查找，并将匹配到的关键词以指定背景色高亮标记，同时输出查找结果汇总。

---

## 二、功能需求详细说明

| F-05 | 提供"全字匹配"复选框选项，默认不启用 | P1 |
| F-06 | 支持正则表达式模式输入（可选开关） | P2 |

| 编号 | 需求描述 | 优先级 |

| F-07 | 查找范围覆盖当前活动文档的全部内容，包括正文、页眉、页脚、脚注、尾注、文本框 | P0 |

### 2.3 背景色标记

| 编号 | 需求描述 | 优先级 |
|------|----------|--------|
| F-10 | 匹配到的关键词自动以背景色高亮标记（Highlight） | P0 |
| F-11 | 默认高亮颜色为黄色（`WdColorIndex.wdYellow`） | P0 |
| F-12 | 提供颜色选择下拉框，支持至少以下预设颜色：黄色、浅绿、浅蓝、粉红、浅橙 | P1 |
| F-13 | 多个关键词可分别指定不同的标记颜色（颜色映射表） | P2 |

## 五、技术实现要点

### 5.1 核心 API

- `Word.Document.Content.Find` — 使用 Word 内置 Find 对象执行全文查找
- `Word.Range.HighlightColorIndex` — 设置匹配文本的背景高亮色
- `Word.Selection.Find.Execute` — 执行查找并定位

### 5.2 关键实现逻辑

```
伪代码:
foreach (keyword in keywords)
{
    Find find = document.Content.Find;
    find.Text = keyword;
    find.MatchCase = isCaseSensitive;
    find.MatchWholeWord = isWholeWord;
    find.Forward = true;

    while (find.Execute())
    {
        // 标记背景色
        find.Parent.HighlightColorIndex = highlightColor;

        // 记录位置信息（页码、段落号）
        results.Add(new SearchResult(keyword, pageNumber, paragraphNumber));
    }
}
```

### 5.3 清除标记逻辑

- 遍历全文所有 Range，将 `HighlightColorIndex` 重置为 `wdNoHighlight`
- 或使用 Word 内置查找替换功能，查找任意高亮文字并清除高亮

## 六、预置关键词参考（可选功能）

根据本项目的科技论文写作规范，可预置以下常用查找关键词类别：

| 类别 | 示例关键词 |
|------|-----------|
