# 科技论文中角标字符与排版上标/下标的规范

> 适用场景：英文科技论文、中文科技论文、中英混排手稿、化学式、单位表达、数学公式、图注与坐标轴标注、参考文献、Word/LaTeX/Origin 排版校对。  
> 核心原则：**排版角标（Word 下标/上标格式或 LaTeX 数学模式）是论文发表的标准做法；Unicode 角标字符仅适用于纯文本环境，不应混入正式排版中。**

---

## 1. 总览：两类"角标"的本质区别

在科技论文写作中，上标（superscript）和下标（subscript）有两种实现方式：

| 类型 | 说明 | 示例（屏幕显示） | 适用场景 | 是否推荐正式论文使用 |
|------|------|:-:|------|:----:|
| **排版原生角标** | Word 字体格式设置的上标/下标，或 LaTeX 数学/文本模式的角标 | $T_c$、$E_g$、$\mathrm{cm}^{-1}$、H₂O | 论文正文、公式、图注、投稿 PDF | ✅ **强烈推荐** |
| **Unicode 角标字符** | 使用 Unicode 预定义的角标码点（如 ₂ ³ ⁻¹ ᵗʰ） | T_c、E_g、cm⁻¹、H₂O | 纯文本环境、注释、文件名、微信群/论坛交流 | ⚠️ 仅限非正式场景 |

**关键问题：** 许多作者在 Word 或 LaTeX 中习惯偷懒，直接用 Unicode 角标字符代替排版原生角标，导致出版后的 PDF 中角标位置异常、字体不匹配、搜索不可靠等问题。

---

## 2. Unicode 角标字符的典型案例

### 2.1 化学式中的数字下标

**常见使用 Unicode 下标字符的写法（不推荐用于正式论文）：**

```
MoSe₂
WSe₂
SiO₂
H₂O
CO₂
CH₄
SO₄²⁻
Fe₂O₃
CuInP₂S₆
hBN
```

> 其中的 `₂` `₃` `₄` `₆` 是 Unicode 下标数字，而非排版下标。

**推荐在论文中使用的写法：**

```text
MoSe₂（Word: 将"2"设为下标格式）
WSe₂（Word: 将"2"设为下标格式）
SiO₂（Word: 将"2"设为下标格式）
H₂O（Word: 将"2"设为下标格式）
```

```latex
\ce{MoSe2}
\ce{WSe2}
\ce{SiO2}
\ce{H2O}
% 或用普通数学模式：
$\mathrm{MoSe_2}$
$\mathrm{SiO_2}$
$\mathrm{H_2O}$
```

**注意事项：**
- 化学式中的数字下标代表原子个数，应使用排版下标，而非 Unicode 预定义下标字符；
- LaTeX 中推荐使用 `\ce{}`（mhchem 宏包）或 `\mathrm{}`+`_`；
- Word 中选中数字后设置"下标"格式，不要从其他字符集粘贴。

---

### 2.2 单位中的负指数

**常见使用 Unicode 上标字符的写法（不推荐）：**

```
cm⁻¹
cm⁻²
s⁻¹
m⁻²
eV⁻¹
K⁻¹
mol⁻¹·L
m⁻³
```

**推荐在论文中使用的写法：**

```latex
$\mathrm{cm}^{-1}$
$\mathrm{cm}^{-2}$
$\mathrm{s}^{-1}$
$\mathrm{m}^{-2}$
$\mathrm{eV}^{-1}$
$\mathrm{K}^{-1}$
$\mathrm{mol}^{-1}\cdot\mathrm{L}$
$\mathrm{m}^{-3}$
```

在 Word 中：`cm⁻¹` 应写为 `cm` + `-1`（选中"-1"设为上标格式）。

**负指数与分数线的选择：**

| 场景 | 推荐写法 | 说明 |
|------|----------|------|
| 单位中 | `$\mathrm{cm}^{-1}$` | 负指数是单位表达的标准形式 |
| 物理量中 | `$\lambda^{-1}$` | 变量可用负指数或分数，注意全文统一 |
| 大段文字中 | `per cm` 或 `$\mathrm{cm}^{-1}$` | 正文中可写"per cm"避免公式打断文本流 |

---

### 2.3 变量中的上标（指数/序号）

**Unicode 字符不推荐：**

```
T₀（Unicode subscript 0）
T₀（Unicode subscript 0）
E⁰（Unicode superscript 0）
x²（如果"²"是 Unicode superscript 2 而非排版上标）
```

**推荐写法：**

```latex
$T_0$
$T_0$  % T下标0，0为正体数字
$E^0$  % E上标0
$x^2$
```

**注意：**
- 即使指数是数字 2 或 3，也应使用排版上标而非 Unicode 角标字符；
- Unicode 中的 `²` (U+00B2) 和 `³` (U+00B3) 在某些字体中可能与正文数字不匹配。

---

### 2.4 参考文献引用中的上标或角括号

某些期刊使用上标编号引用参考文献：

```
Text text text [1,2].
```

统一使用排版上标或方括号，不要将 Unicode 上标数字 `¹²³` 混入正文。

**不推荐：**

```
Text text text ¹'²'³.
```

**推荐（按期刊要求）：**

```text
Text text text [1,2,3].  % 方括号格式
```

或：

```text
Text text text¹,²,³（Word: 将数字设为上标格式）
```

## 3. 常见误区：在不恰当的场合混用 Unicode 角标

### 3.1 图注和坐标轴标签中的角标

**不推荐（在 Origin/Matlab 中直接粘贴 Unicode 角标）：**

```
Energy (cm⁻¹)
Gate voltage, Vg (V) ← 下标"g"也应排版，不宜用 Unicode 字符
PL Intensity (a.u.)
```

**推荐（使用排版软件的原生角标功能）：**

在 Origin 中：使用文本工具中的上标/下标功能或 LaTeX 模式。

```
Energy (cm⁻¹)
Gate voltage, Vg (V)
PL Intensity (a.u.)
```

### 3.2 方法名称中的角标

**不推荐：**

```
CVD-grown SE₂  ← 第二层似乎使用的是 Unicode 下标"₂"
FET
```

**推荐：**

```text
MoSe₂（排版下标）
WSe₂（排版下标）
FET
```

### 3.3 表格中的角标

表格中的数据单位和符号同样应使用排版角标，而不是 Unicode 角标字符。

**不推荐：**

| Parameter | Value |
|-----------|-------|
| $T_c$ | 150 K | ← 排版写 T_c 时要注意全文统一 |

## 4. 区分：何时可以使用 Unicode 角标字符？

| 场景 | 是否建议使用 Unicode 角标 | 原因 |
|------|:-------:|------|
| 论文正文（Word/LaTeX） | ❌ 不建议 | 应使用排版原生角标 |
| 论文正文（纯文本 .txt） | ✅ 可以 | 纯文本无格式支持 |
| 邮件正文 | ✅ 可以 | 纯文本环境下角标字符是唯一选择 |
| 微信/QQ/社交媒体 | ✅ 可以 | 非正式交流，追求可读性 |
| 文件名/文件夹名 | ✅ 可以 | 避免特殊字符冲突 |
| 源代码注释 | ✅ 可以 | 纯文本上下文 |
| Origin/MS Office 图表 | ❌ 不建议 | 应使用软件内上标/下标功能 |
| 投稿 PDF 生成前 | ❌ 绝对不建议 | 容易在 PDF 渲染中出现异常 |

## 5. Unicode 角标字符速查表

### 5.1 Unicode 下标数字

| 字符 | Unicode | 说明 | 常见误用场景 |
|:----:|:-------:|------|:----:|
| ₀ | U+2080 | 下标 0 | $T_0$、$E_0$ |
| ₁ | U+2081 | 下标 1 | $T_1$、$n_1$ |
| ₂ | U+2082 | 下标 2 | $\mathrm{MoSe_2}$、$x_2$ |
| ₃ | U+2083 | 下标 3 | $\mathrm{Fe_2O_3}$ |
| ₄ | U+2084 | 下标 4 | $\mathrm{CH_4}$ |
| ₅ | U+2085 | 下标 5 | $\mathrm{N_2O_5}$ |
| ₆ | U+2086 | 下标 6 | $\mathrm{CuInP_2S_6}$ |
| ₓ | U+2093 | 下标 x | $k_x$、$q_x$ |

### 5.2 Unicode 上标字符

| 字符 | Unicode | 说明 | 常见误用场景 |
|:----:|:-------:|------|:----:|
| ⁻ | U+207B | 上标负号 | $\mathrm{cm}^{-1}$ |
| ⁺ | U+207A | 上标正号 | $\mathrm{Ca}^{2+}$ |
| ¹ | U+00B9 | 上标 1 | $\mathrm{s}^{-1}$ 中的 1 |
| ² | U+00B2 | 上标 2 | $x^2$、$\mathrm{cm}^{-2}$ |
| ³ | U+00B3 | 上标 3 | $x^3$、$\mathrm{m}^{-3}$ |
| ⁱ | U+2071 | 上标 i | 极少用 |
| ⁺² | 组合 | 无法用单一 Unicode 实现 | $\mathrm{Ca}^{2+}$ 需 2 个字符 |

### 5.3 Unicode **无法实现**的常见角标需求

| 需求 | 无法用 Unicode 实现的原因 | 必须使用排版原生角标 |
|------|-------------------------|:----:|
| 复杂上标如 `$X_\mathrm{T}^{*}$` | Unicode 没有"星号上标"和"T下标"的组合 | ✅ |
| 多字符上下标如 `$k_\mathrm{B}$`、`$V_\mathrm{ds}$` | Unicode 只有单字符下标 | ✅ |
| 同时有上标和下标的符号如 `$I_{X_\mathrm{A}}$` | Unicode 不支持嵌套角标 | ✅ |
| 带负指数的单位如 `$\mathrm{cm}^{-1}$` | Unicode 上标负号可能变形 | ✅（建议） |
| 斜体角标如 `$x_n$` 中的 `n` | Unicode 角标没有斜体版本 | ✅ |

## 6. 常见错误清单

### 错误 1：化学式中混用 Unicode 下标

**错误：**
```text
MoSe₂ was exfoliated onto SiO₂/Si substrate.
```
← `₂` 和 `₂` 是 Unicode 字符，在 PDF 中可能与正文数字字体不一致

**正确：**
```text
MoSe₂ was exfoliated onto SiO₂/Si substrate.
```
← 在 Word 中将"2"设为下标格式；在 LaTeX 中使用 `$\mathrm{MoSe_2}$` 或 `\ce{MoSe2}`

---

### 错误 2：单位中使用 Unicode 负指数

**错误：**
```text
Raman shift (cm⁻¹)
```
← `⁻` 和 `¹` 是 Unicode 上标字符，与正文字体不匹配

**正确：**
```latex
Raman shift ($\mathrm{cm}^{-1}$)
```
或在 Word 中：`cm` + 空格 + `-1`（"-1"设上标格式）

---

### 错误 3：变量下标混用 Unicode 字符和排版下标

**错误：**
```text
T_c = 150 K（其中下标 c 是 Unicode 下标字符）
V_g = −4 V（其中下标 g 是 Unicode 下标字符）
```
← 同一篇论文中有的下标是排版格式、有的是 Unicode 字符，视觉不一致

**正确：**
全文统一使用排版原生下标，不要混用。

---

### 错误 4：从网页直接复制粘贴角标

**错误：**
从维基百科、其他 PDF 或网页复制化学式或单位时，无意中带入了 Unicode 角标字符。

**正确：**
粘贴后检查是否带有 Unicode 角标字符。在 Word 中，可以显示隐藏符号/格式标记来识别。

---

### 错误 5：坐标轴/图例中的角标混合

**错误：**
在 Origin 中某个图标注解说使用了排版下标，同一个图的另一个标签却用了 Unicode 下标字符。

**正确：**
统一使用 Origin 的文本格式工具中的上标/下标功能（插入→上标/下标，或使用富文本格式）。

## 7. 投稿前检查清单

### 7.1 Word 用户检查方法

- [ ] 打开"显示/隐藏编辑标记"（¶ 按钮），查看化学式和单位中是否有异常字符
- [ ] 用搜索功能查找 Unicode 角标字符（可复制一个 Unicode 下标数字如 `₂` 到搜索框，逐一检查）
- [ ] 重点检查的区域：化学式（MoSe₂、WSe₂、SiO₂）、单位（cm⁻¹、s⁻¹）、温度（T_c、T_0）
- [ ] 确认所有上标/下标都是通过 Word 字体格式设置的，而非从外部粘贴的 Unicode 字符

### 7.2 LaTeX 用户检查方法

- [ ] 搜索 `.tex` 文件中是否包含 Unicode 角标字符（可能是从文本编辑器或浏览器粘贴导致的）
- [ ] 使用脚本或编辑器正则功能查找非 ASCII 字符中的上下标范围：
  - 下标范围：U+2080–U+209F
  - 上标范围：U+2070–U+207F、U+00B2、U+00B3、U+00B9
- [ ] 确认所有化学式使用 `\ce{}` 或 `\mathrm{}`+`_`，而非 Unicode 预定义字符
- [ ] 确认所有单位的负指数使用 `^{-1}` 而非 Unicode `⁻¹`

### 7.3 通用检查方法

- [ ] 在 PDF 预览中检查化学式和单位中的数字字体是否与正文一致
- [ ] 在 PDF 中搜索化学式（如搜索 "MoSe2"），确认下标的数字是否被正确索引
- [ ] 检查图注和坐标轴标签中的角标是否与正文格���一致
- [ ] 检查 Origin 导出的图片中是否不小心包含了 Unicode 角标字符

## 8. 最简速记规则

> 如果只能记住 5 条：

1. **写论文时永远不要从外部文档复制粘贴角标字符**——你大概率复制了 Unicode 角标而非排版原生角标
2. **化学式中的数字下标** → 在 Word 中选中数字设"下标"，在 LaTeX 中用 `_`
3. **单位的负指数** → 在 LaTeX 中用 `^{-1}`，在 Word 中上标格式化 `-1`
4. **图注/坐标轴中的角标** → 使用 Origin 或作图软件的原生角标功能，不要粘贴 Unicode 字符
5. **预览 PDF 时检查**：角标数字的字体是否与正文数字一致——如果不一样，说明混入了 Unicode 角标

## 9. 建议的全文统一方案

| 场景 | ✅ 推荐方案（排版原生角标） | ❌ 不要用（Unicode 角标字符） |
|------|:----:|:----:|
| MoSe₂ 化学式 | Word: `MoSe`+"2"下标；LaTeX: `\ce{MoSe2}` 或 `$\mathrm{MoSe_2}$` | `MoSe₂`（Unicode U+2082） |
| cm⁻¹ 波数 | Word: `cm`+"-1"上标；LaTeX: `$\mathrm{cm}^{-1}$` | `cm⁻¹`（Unicode U+207B + U+00B9） |
| T_c 临界温度 | Word: `T`+"c"下标；LaTeX: `$T_\mathrm{c}$` | `Tₓ`（Unicode U+2093） |
| V_g 栅压 | Word: `V`+"g"下标；LaTeX: `$V_\mathrm{g}$` | `V_g`（Unicode U+2092） |
| E_g 带隙 | Word: `E`+"g"下标；LaTeX: `$E_\mathrm{g}$` | `E_g`（Unicode U+2092） |
| x² 平方 | Word: `x`+"2"上标；LaTeX: `$x^2$` | `x²`（Unicode U+00B2） |
| 参考文献上标 | Word: 选中数字设上标；LaTeX: `\textsuperscript{1}` | `¹` `²` `³`（Unicode） |
| s⁻¹ 单位 | Word: `s`+"-1"上标；LaTeX: `$\mathrm{s}^{-1}$` | `s⁻¹`（Unicode） |

---

## 10. 最后建议

在物理和材料科学论文中，上标和下标几乎无处不在——化学式、单位、变量标注、峰标签、参考文献。一个看似微小的疏忽（从网页复制了一个 Unicode 角标字符）就可能导致整篇 PDF 的字体渲染不统一。现代排版工具（Word、LaTeX、Origin）都提供了完善的上标/下标功能，没有理由使用替代字符。

**方法极为简单：** 每次在文档中输入任何带角标的符号前，提醒自己——"这是排版文件，不是纯文本"。花 5 秒钟用 Word 的下标按钮或 LaTeX 的 `_` 和 `^`，换来的是全文统一的专业排版质量。

这不是一个"哪种更正确"的问题，而是一个 **"哪种不会让审稿人和排版编辑觉得奇怪"** 的问题。
